# Poetry Analysis Tool

## Overview

The Poetry Analysis Tool is a Python program designed to allow users to explore and analyze the poetry of three renowned American poets: Sylvia Plath, Wallace Stevens, and Robert Frost. Users can perform various statistical analyses on the selected poet's work, including word counts, frequency analysis of different parts of speech, and context exploration for specific words.

## Features

- Choose from three poets: Sylvia Plath, Wallace Stevens, and Robert Frost.
- Analyze the total number of words and unique words in the selected poet's collection.
- View the most frequent words, nouns, adjectives, and verbs in the text.
- Explore the contexts in which specific words occur.
- Easy navigation to choose a different poet or exit the program.

## Requirements

To run this program, you need:

- Python 3.x
- The following Python libraries:
  - textblob
  
You can install the required libraries using pip:

bash
pip install textblob

## Getting Started

1. Clone this repository to your local machine

2. Ensure you have the text files for each poet's collection:
   - Plath.txt
   - Stevens.txt
   - Frost.txt

   Place these files in the same directory as the Python script.

3. Run the program:
bash
   python poetry_analysis.py

4. Follow the on-screen prompts to select a poet and perform analyses.

## Usage

Upon running the program, you will be greeted with options to choose a poet. Once a poet is selected, you can request various analyses, such as:

- Total word count
- Unique word count
- Most frequent words
- Most frequent nouns, adjectives, or verbs
- Contexts for specific words

You can also return to the main menu to select a different poet or exit the program.

## Example

Here is an example of how to use the program:

1. Select "Sylvia Plath".
2. Choose to view the most frequent words.
3. Enter a specific word to see its context in the text.
